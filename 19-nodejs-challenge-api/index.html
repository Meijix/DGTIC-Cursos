Node.js — Challenge API

Proyecto integrador: aplicacion CLI y servidor Express con API REST en Node.js.

Sobre este proyecto

Challenge de Node.js que combina una aplicacion de linea de comandos (CLI) con un servidor Express que expone una API REST. Incluye manejo de rutas, middleware, JSON y paquetes npm. El proyecto demuestra el uso practico de Node.js tanto para scripts como para servicios web.
