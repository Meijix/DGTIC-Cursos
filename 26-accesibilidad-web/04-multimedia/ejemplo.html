Galeria de Medios Accesible

Cada elemento multimedia tiene alternativas de texto adecuadas. Revisa el codigo fuente para ver los comentarios explicativos.

Imagen informativa con datos

El alt text describe los datos esenciales del grafico. El figcaption proporciona contexto adicional visible para todos.

Tecnica: alt describe el contenido visual. figcaption anade interpretacion. Ambos coexisten.

Icono decorativo junto a texto

El icono SVG es decorativo porque el texto ya describe la accion. Se usa alt="" (o aria-hidden en SVG inline) para que el lector lo ignore.

Descargar informe en PDF
Tecnica: aria-hidden="true" en el SVG. El lector de pantalla solo anuncia "Descargar informe en PDF, enlace".

Imagen como enlace

Cuando una imagen es un enlace, el alt describe a donde lleva, no como se ve la imagen.

Tecnica: El alt dice "Ir a la pagina principal de MiSitio" (destino), no "Logo azul con texto MiSitio" (aspecto).

Imagen compleja con descripcion larga

Para diagramas complejos, el alt es breve y se ofrece una descripcion detallada por separado.

Descripcion detallada del diagrama

El diagrama muestra tres componentes conectados en secuencia:

1. Cliente (caja azul): Navegador web del usuario. Se conecta al servidor mediante protocolo HTTPS.

2. Servidor (caja verde): Aplicacion backend que procesa las peticiones. Se comunica con la base de datos mediante consultas SQL cifradas.

3. Base de datos (caja naranja): Almacena y gestiona los datos persistentes del sistema.

El flujo es unidireccional: cliente envia peticion al servidor, el servidor consulta la base de datos, y la respuesta regresa por el mismo camino.

Tecnica: alt breve + details/summary para la descripcion completa. Accesible para lectores de pantalla y util para todos.
