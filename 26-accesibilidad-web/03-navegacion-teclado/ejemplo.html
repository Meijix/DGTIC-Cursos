Focus Visible

Navega con Tab y observa el outline azul en cada elemento interactivo. Nunca se pierde el indicador de foco.

Modal con Focus Trap

Al abrir el modal, el foco se mueve dentro. Tab cicla entre los elementos del modal. Escape cierra y regresa el foco.

Tabs con Roving Tabindex

Tab entra y sale del widget. Usa flechas izquierda/derecha para cambiar entre pestanas.

HTML proporciona la estructura semantica del documento. Usa elementos como header, nav, main y footer para crear landmarks accesibles.

CSS controla la presentacion visual. Es crucial para el contraste de color, los estilos de foco visible y los disenos responsivos accesibles.

JavaScript maneja la interactividad: focus management, ARIA dinamico, live regions y el comportamiento de widgets como tabs y modales.
