# DGTIC - Curso de HTML y CSS

Ejercicios y proyectos del curso de HTML y CSS impartido por la DGTIC (UNAM), complementados con proyectos propios para reforzar los conceptos aprendidos.

**Autora:** Natalia Edith Mejia Bautista

---

## Contenido del curso

El repositorio esta organizado en modulos progresivos, desde los fundamentos hasta proyectos completos:

| # | Modulo | Temas principales |
|---|--------|-------------------|
| 01 | [Fundamentos HTML](01-fundamentos-html/) | Estructura semantica HTML5, formularios, listas de definicion |
| 02 | [Fundamentos CSS](02-fundamentos-css/) | Variables CSS, box model, colores, tipografia |
| 03 | [Pagina web basica](03-pagina-web-basica/) | Sitio multi-pagina, enlaces, imagenes, formularios |
| 04 | [Flexbox - Dados](04-flexbox-dados/) | Display flex, justify-content, align-items, flex-direction |
| 05 | [Componentes CSS](05-componentes-css/) | Tarjetas con gradientes, box-shadow, diseno de componentes |
| 06 | [Responsive - Fundamentos](06-responsive-fundamentos/) | Media queries, mobile-first, anchos flexibles |
| 07 | [Responsive - CSS Grid](07-responsive-css-grid/) | CSS Grid, grid-template-columns, layouts multi-columna |
| 08 | [Proyecto: Sitio de servicios](08-proyecto-web-servicios/) | Proyecto integrador con gradientes, cards responsivas, multiples breakpoints |
| 09 | [Portfolio profesional](09-portfolio-profesional/) | Portfolio completo con animaciones scroll, navegacion suave, JavaScript |

---

## Tecnologias

- HTML5 (estructura semantica)
- CSS3 (Flexbox, Grid, variables, media queries, gradientes, animaciones)
- JavaScript (animaciones de scroll, navegacion interactiva)

## Como usar

Cada carpeta es un modulo independiente. Abre el archivo `.html` principal en tu navegador para ver el resultado.

```bash
# Ejemplo: abrir el modulo de Flexbox
open 04-flexbox-dados/dados.html
```
